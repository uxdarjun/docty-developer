Option Type Description Value
Gender Optional You can generate the patients list by the gender using the the acceptable values as below
Acceptable values : Male, Female, Other
City Code Optional To get the patient list by the area you can add the multiple city code seprated by the (,) comma.
Example: 12541, 4541, 1245
Medical Records Optional To generate the patient list by the medical records use the below parameters
BMI: UnderWeight, OverWeight, Normal
Blood Groups: A, A+, B etc..
